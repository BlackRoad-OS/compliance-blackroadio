## ‚úÖ Qualified Custodian Requirements

### SEC Custody Rule Standards
A qualified custodian must be:
1. **Bank or savings association** (FDIC insured)
2. **Registered broker-dealer** (FINRA member)
3. **Registered futures commission merchant** (CFTC)
4. **Foreign financial institution** (if holds client assets in jurisdiction)

### Additional Requirements for Crypto
- **Cold storage:** 95%+ of assets offline
- **Multi-signature wallets:** 3-of-5 minimum
- **Insurance:** Crime/specie coverage
- **SOC 2 Type II:** Annual attestation
- **Disaster recovery:** Geographically distributed keys
